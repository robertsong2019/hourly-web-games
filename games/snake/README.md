# 🐍 贪吃蛇

经典贪吃蛇网页游戏，纯 HTML/CSS/JavaScript 实现。

## 🎮 游戏特性

- ✅ 经典贪吃蛇玩法
- ✅ 键盘方向键控制
- ✅ 移动端触控支持
- ✅ 分数和最高分记录
- ✅ 响应式设计
- ✅ 暂停/继续功能

## 🕹️ 如何玩

### 电脑端
使用键盘方向键 ↑↓←→ 控制蛇的移动方向

### 手机端
点击屏幕上的方向按钮控制

## 🎯 游戏规则

1. 蛇会自动移动
2. 吃到红色食物，蛇身变长，得 10 分
3. 撞墙或撞到自己，游戏结束
4. 尽量获得高分！
