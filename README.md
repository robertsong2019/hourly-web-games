# 🎮 Hourly Web Games

每小时自动生成一个网页小游戏的仓库。

## 📖 项目介绍

这是一个自动化游戏开发项目，每小时会自动生成一个新的网页小游戏。所有游戏都是纯 HTML/CSS/JavaScript 实现，无需后端，可以直接在浏览器中运行。

## 🎯 项目目标

- ⏰ **自动化**: 每小时自动生成一个新游戏
- 🎮 **可玩性**: 每个游戏都经过自测，确保基本可用
- 📚 **多样性**: 涵盖各种类型的休闲小游戏
- 🔧 **开源**: 所有代码开源，可供学习和参考

## 🚀 如何运行

### 在线访问

每个游戏都可以直接在浏览器中打开：

```
https://robertsong2019.github.io/hourly-web-games/games/[游戏名称]/
```

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/robertsong2019/hourly-web-games.git

# 打开任意游戏
open games/[游戏名称]/index.html
```

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript** - 游戏逻辑
- **Canvas API** - 图形渲染（部分游戏）

## 🤖 自动化流程

1. **触发**: 每小时整点触发（OpenClaw Cron）
2. **生成**: 使用 Claude Code + GLM-5 自动生成游戏
3. **测试**: 自动运行基本功能测试
4. **提交**: 通过测试后提交到仓库
5. **部署**: 自动部署到 GitHub Pages
